Welcome to NMPZ!

This challenge uses SSS to encode the flag. Therefore, you only have to solve 9/10 challenges for each set. You need to use this script to combine your shares into a flag.

The script will be updated if we drop the difficulty to 8/10 in the future. There will be an announcement if that happens.

Guesses need to be within 50m of the location.

Beware sometimes the compass is inverted or wrong, sorry.

This challenge is best played on a Chromium based browser.
